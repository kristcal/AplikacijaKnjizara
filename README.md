# Aplikacija Knjizara

Ovo je web aplikacija za knjižaru, razvijena kao **studentski projekat za predmet Napredne Java Tehnologije**.  
Backend je napravljen u **Java Spring Boot**, a frontend u **React**.

## Funkcionalnosti
- Pregled i pretraga knjiga
- Upravljanje korisnicima (registracija, prijava)
- Porudžbine i praćenje narudžbina
- Administracija knjiga i korisnika (samo za admina)

## Tehnologije
- **Backend:** Java, Spring Boot, Maven
- **Frontend:** React, HTML, CSS, JavaScript
- **Baza podataka:** MySQL
- **Alati:** NetBeans, Visual Studio Code

## Instalacija i pokretanje

Klonirajte repozitorijum i pokrenite backend u NetBeans-u, a frontend u Visual Studio Code-u koristeći `npm install` i `npm start`.  
Frontend će biti dostupan na `http://localhost:3000`, a backend na `http://localhost:8080`.
